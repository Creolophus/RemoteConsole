iOS远程日志操作手册

Usage

  1. 使用iPhone安装8.3以后的测试版Onstar App
  2. iPhone连接WiFi => 233333333 (之后可能会改名)
  3. 确认该台iPhone已在陶量处登记，可在10.16.142.177:8088/userlist.html确认。
  4. 打开Onstar App
  5. 打开浏览器，在地址栏输入10.16.142.177:8088/?user=#name#，#name#为登记过的该台iPhone的名称
  6. 正常情况下，浏览器会显示Connect，代表已经连接成功。

PS.

  1. 由于网络原因，IP地址（即上文中的10.16.142.177）可能会有变化，每天会用脚本发送Email给各位本机的最新IP，如果输入地址后，浏览器中什么内容都没，很大可能是IP地址换了。
  2. 现可以在APP"关于"界面，猛烈摇动手机查看该版本App是否支持远程Log
  3. 如需在生产环境查看日志，需要另外打包安装。

颜色示例

log颜色 代表含义
系统log 系统log\SDK的log，可点击右下角按钮关闭
HTTP Request 客户端像Onstar服务器发送的网络请求
HTTP Response Onstar后台返回的数据
HTML5加载 客户端加载HTML页面信息
dealloc 客户端viewWillAppear或者dealloc
代表检测到error或者exception字段，错误信息
代表检测到warning字段，警告信息
云音乐中socket长连接信息
